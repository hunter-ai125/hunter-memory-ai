# 🦅 捕猎式记忆 AI（Hunter Memory）

> 一个让记忆像生物一样“捕食、进化、休眠”的 AI 记忆系统实验。

## 🧬 这是什么？
这不是普通的向量数据库。每条记忆都自带一个轻量级神经网络，它们会通过 **“对战合并”** 优胜劣汰，高等级的记忆会变成 **“捕猎者”**，长期不用的会自动 **“冻结休眠”**。

## 🤖 代码来源
**本项目 100% 的代码由 DeepSeek AI 编程助手生成**，我担任概念设计与产品测试。 
这是一个“人类创意 + AI 工程”的协作实验。

## ✨ 核心特性
- ⚔️ **模型融合进化**：记忆块通过神经网络权重合并实现“捕食”。
- ❄️ **自动冻结/解冻**：3分钟无活动自动休眠，查询命中即刻唤醒。
- 🧠 **本地大模型支持**：集成 Ollama / OpenAI，支持 RAG 检索增强。
- 🎮 **沉浸式终端交互**：自带聊天界面，看着 AI 生态在眼前进化。

## 🚀 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 确保已安装 Ollama 并拉取模型：`ollama pull qwen2.5:1.5b`
3. 运行聊天：`python chat.py`
